# Roguelike Run Mode - Game Design Document

## Overview

Roguelike Run — это режим прогрессии, где игроки проходят забеги с целью набрать 9 побед до 4 поражений. Игроки строят колоду через систему фракций, лидеров и драфта.

---

## Основные отличия от MVP

| Аспект | MVP | Roguelike Run |
|--------|-----|---------------|
| Фракции | 1 (общая) | 6 фракций, 25 юнитов каждая |
| Лидеры | Нет | 18 лидеров (3 на фракцию) |
| Заклинания | Нет | 2 заклинания в колоде |
| Размер колоды | N/A | 14 карт (12 юнитов + 2 заклинания) |
| Сетка | 8×10 | 8×2 (зона высадки) |
| Бюджет | 30 фиксированный | 10g → 65g прогрессия |
| Тиры юнитов | Нет | T1 → T2 → T3 апгрейды |
| Режим игры | Одиночный бой | Забег 9 побед / 4 поражения |
| Матчмейкинг | PvP очередь | Асинхронный vs снапшоты |

---

## Боевые механики

Roguelike режим использует **Core Mechanics 2.0** — расширенную систему боевых механик, которая добавляет глубину и стратегичность к боям.

### Фазы боя и механики

Каждый раунд боя проходит через несколько фаз, на каждой из которых применяются определённые механики:

| Фаза | Механики | Описание |
|------|----------|----------|
| **TURN_START** | Resolve, Ammunition, Aura, Phalanx | Восстановление ресурсов, обновление бонусов |
| **MOVEMENT** | Engagement, Intercept, Overwatch, Charge | Движение, зона контроля, перехват |
| **PRE_ATTACK** | Facing, Flanking, Charge, LoS, Ammunition | Проверка возможности атаки |
| **ATTACK** | Armor Shred, Riposte, Contagion | Выполнение атаки и эффектов |
| **POST_ATTACK** | Resolve, Phalanx | Проверка боевого духа, обновление формирования |
| **TURN_END** | Contagion, Aura, Overwatch | Распространение эффектов, очистка состояния |

### Включённые механики (Phase 1)

В Phase 1 (Humans vs Undead) включены следующие механики:

#### Tier 0: Базовые
- **Facing** — Юниты имеют направление (N/S/E/W), влияет на урон и контратаки

#### Tier 1: Основной боевой слой
- **Resolve (Боевой дух)** — Система морали, юниты отступают/рассыпаются при resolve = 0
  - Максимум: 100
  - Восстановление: +5 за ход
  - Урон морали: 25% от ATK атакующего
  - Сопротивление: уменьшает урон морали на процент
  - Люди отступают, Нежить рассыпается
- **Engagement (Зона контроля)** — Юниты в ближнем бою не могут свободно двигаться
  - Лучники получают штраф -50% урона в ближнем бою
- **Flanking (Фланговая атака)** — Атаки с фланга/тыла наносят больше урона
  - Фланг: +15% урона + 12 урона боевому духу
  - Тыл: +30% урона + 20 урона боевому духу

#### Tier 2: Продвинутые
- **Riposte (Контратака)** — Защитник может контратаковать при фронтальной атаке
  - Шанс зависит от инициативы
  - Контратака наносит 50% урона
- **Intercept (Перехват)** — Пехота может перехватить движущихся врагов
  - Мягкий перехват: пехота замедляет проходящих врагов
  - Стоимость отхода: 2 клетки движения
- **Aura (Аура)** — Пассивные эффекты от лидеров и юнитов

#### Tier 3: Специализированные
- **Charge (Атака с разбега)** — Кавалерия получает бонус урона за расстояние
  - +20% урона за каждую клетку движения (макс +100%)
  - Минимум 3 клетки для активации
- **Overwatch (Дозор)** — Лучники могут стрелять при движении врагов
- **Phalanx (Фаланга)** — Формирование даёт бонусы броне и боевому духу
  - +1 броня за каждого соседа (макс +5)
  - +5 боевого духа за каждого соседа (макс +25)
- **Line of Sight** — Лучники должны видеть цель
  - Прямой огонь: блокируется юнитами
  - Дуговой огонь: игнорирует препятствия, но -20% точность
- **Ammunition** — Лучники имеют боеприпасы, маги — перезарядку
  - Боеприпасы: 6 выстрелов
  - Перезарядка магов: 3 раунда

#### Tier 4: Контр-механики
- **Contagion (Заражение)** — Эффекты распространяются на соседних юнитов
  - Огонь: 50% шанс
  - Яд: 30% шанс
  - Проклятие: 25% шанс
  - Мороз: 20% шанс
  - Чума: 60% шанс
  - Бонус в фаланге: +15%
- **Armor Shred (Разрушение брони)** — Физические атаки ослабляют броню
  - -1 броня за атаку (макс -40% от базовой)

### Механики, отключённые в Phase 1

Следующие механики планируются для Phase 2+:
- Дополнительные эффекты Resolve
- Расширенные Aura типы
- Специальные взаимодействия между фракциями

### Интеграция с лидерами

Лидеры могут иметь пассивные способности, которые взаимодействуют с механиками:
- Пассивка может давать бонусы к Resolve
- Пассивка может улучшать Phalanx формирования
- Пассивка может изменять Charge параметры

Подробнее см. раздел **Лидеры** ниже.

## Драфт

### Начальный драфт

3. Карты идут в руку

|---------|--------|
| Победа | 7g базово |

| Серия побед (3+) | +2g за каждую победу в серии |

### Формула награды
```typescript
// Starting gold: 10g
// Win reward: +7g + streak bonus
// Lose reward: +9g (catch-up mechanic)

function getGoldReward(won: boolean, winStreak: number): number {
  if (!won) return 9;  // Lose: 9g (catch-up mechanic)
  
  const base = 7;  // Win: 7g base
  const streakBonus = winStreak >= 3 ? (winStreak - 2) * 2 : 0;
  return base + streakBonus;
}
```

## Асинхронный PvP

### Система снапшотов
1. После каждого боя сохраняется снапшот команды
2. Матчмейкинг против снапшотов других игроков
3. Детерминированная симуляция (один seed = один результат)

### Алгоритм матчмейкинга
```typescript
async function findOpponent(run: Run): Promise<Snapshot | null> {
  // 1. Найти снапшоты с похожим числом побед
  const candidates = await findByWins(run.wins - 1, run.wins + 1);
  
  // 2. Фильтр по рейтингу (±200)
  const filtered = candidates.filter(s => 
    Math.abs(s.rating - run.rating) <= 200
  );
  
  // 3. Случайный выбор
  if (filtered.length > 0) {
    return randomChoice(filtered);
  }
  
  // 4. Fallback: бот
  return generateBot(run.wins);
}
```

## Рейтинговая система

### Расчёт рейтинга (ELO)
```typescript
function calculateRatingChange(
  playerRating: number,
  opponentRating: number,
  won: boolean
): number {
  const K = 32;
  const expected = 1 / (1 + Math.pow(10, (opponentRating - playerRating) / 400));
  const actual = won ? 1 : 0;
  return Math.round(K * (actual - expected));
}
```

## Стратегические последствия механик

### Позиционирование

С включением **Facing** и **Flanking** позиционирование становится критичным:
- Юниты должны быть развёрнуты лицом к врагу
- Фланговые атаки наносят +15% урона и урон боевому духу
- Тыловые атаки наносят +30% урона и больше урона боевому духу

**Стратегия**: Размещайте танков в центре, чтобы защитить уязвимых юнитов от фланговых атак.

### Формирования

**Phalanx** механика награждает плотные формирования:
- Каждый сосед даёт +1 броня и +5 боевого духа
- Максимум +5 брони и +25 боевого духа в плотном формировании

**Стратегия**: Группируйте танков и поддержку вместе для максимальной защиты.

### Боевой дух (Resolve)

**Resolve** система добавляет новое измерение к боевым потерям:
- Юниты теряют боевой дух от фланговых/тыловых атак
- При resolve = 0 люди отступают, нежить рассыпается
- Боевой дух восстанавливается +5 за ход

**Стратегия**: Фланговые атаки могут быть эффективнее, чем прямые атаки, если они сломают боевой дух врага.

### Кавалерия (Charge)

**Charge** механика делает кавалерию мощной при правильном использовании:
- +20% урона за каждую клетку движения (макс +100%)
- Требует минимум 3 клетки для активации

**Стратегия**: Размещайте кавалерию далеко от врага, чтобы они могли разогнаться и нанести мощный удар.

### Лучники (Ammunition + Overwatch)

**Ammunition** и **Overwatch** делают лучников более интересными:
- 6 боеприпасов на бой
- Могут стрелять при движении врагов (Overwatch)
- Получают штраф -50% в ближнем бою (Engagement)

**Стратегия**: Держите лучников в тылу, защищённых танками. Используйте Overwatch для контроля движения врага.

### Маги (Cooldowns + Contagion)

**Cooldowns** и **Contagion** делают магов опасными в группах:
- Перезарядка 3 раунда между заклинаниями
- Эффекты распространяются на соседей (Contagion)
- Плотные формирования врага = быстрое распространение эффектов

**Стратегия**: Используйте магов для распространения эффектов в плотных формированиях врага.

## Планы на Phase 2+ (Будущие фракции)

### Новые фракции (Phase 2)

Когда будут добавлены новые фракции (Order, Chaos, Nature, Shadow, Arcane, Machine), они будут использовать расширенные боевые механики:

| Фракция | Новые механики | Описание |
|---------|----------------|----------|
| **Order** | Enhanced Phalanx | Улучшенные формирования, +50% бонусы |
| **Chaos** | Enhanced Contagion | Быстрое распространение эффектов |
| **Nature** | Regeneration | Восстановление HP в течение боя |
| **Shadow** | Stealth | Невидимость, скрытые атаки |
| **Arcane** | Mana Pool | Система маны вместо боеприпасов |
| **Machine** | Armor Stacking | Накопление брони, неуязвимость |

### Расширенные механики (Phase 2+)

Планируется добавить новые механики для большей глубины:
- **Energy System** — Юниты накапливают энергию и активируют ульты
- **Morale Cascades** — Цепные реакции боевого духа
- **Terrain Effects** — Влияние местности на боевые механики
- **Weather System** — Погода влияет на эффективность способностей
- **Faction Synergies** — Специальные взаимодействия между фракциями

Все эти механики будут интегрированы через систему **Core Mechanics 2.0** с использованием feature flags.
